Header

After the php open tags there should be a copyright notice identifing Paragon.

<?php

Continuations

All forms of continuation should start on the next line NOT end the previous.

The below example shows how code multi line concat (.) or dot operator

$myString =   'start'   
            . ' put the space on the next line'
            . ' as that too is a form of continuation.'
				; // put the semicolon alone on the last line 
{
	<body of the function>
}
List Commas

when commas are used on long lists for arrays or function parameters which span more than one line the commas should start the continued line 

function myfunc( $param_one
               , $param_two
               , $param_three = <three's default value>
               , $param_four  = <four's default value>
               )
{
	<body of the function>
}

$var = array( 'this' => 'that'
            , 'other' => 101
            , 'last_item' => 'last item'
            );
